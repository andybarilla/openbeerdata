Open Beer Data Project

The Open Beer Data Project aims to provide a free, as in beer and as in speech, way to access information related to beer and homebrewing. This will hopefully serve as a unified place to access data regarding ingredients and other information. The key factor that distinguishes this project from others is that the main code is kept in a repository hosted by Google so that it is not reliant on any one person and it can be maintained even if everyone involves is replaced by replicants.

The open beer database currently includes:
  • Hops - general data
  • Yeast - Wyeast and White Labs
  • Grains/Fermentables - Breiss
  • SRM Colors - srm to rgb mapping
  • BJCP
The data involved, excluding the BJCP, is licensed under the Open Database License and can be used for any purpose, commercial or personal.

The following data formats are currently avaiable:
  • Sqlite - openbeerdata.db
  • CSV files
    • hops.csv
    • fermentables.csv
    • yeast.csv
    • srm.csv

If you have information to provide that is currently not included in the database or would like to keep your data updated regularly, please contact Andrew Barilla. I can work with you to automate the process as best as possible including data conversion.
